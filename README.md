# 🚀 21 Days Machine Learning Challenge  

Welcome to my **21 Days ML Challenge** repository!  
This repo documents my journey of learning and building projects across **Machine Learning, Deep Learning, and Generative AI**.

## 📌 About This Repository  

This repository is a structured collection of projects completed over **21 days**, focusing on building strong fundamentals and practical skills in AI/ML.

The journey is divided into 3 phases:

---

## 🗓️ Week 1: Data Analysis & Machine Learning  
- Exploratory Data Analysis (EDA)  
- Data Cleaning & Preprocessing  
- Feature Engineering  
- Visualization (Seaborn, Matplotlib)  
- Supervised Learning Algorithms  
- Model Evaluation  

---

## 🧠 Week 2: Deep Learning  
- Introduction to Neural Networks  
- Forward & Backpropagation  
- Activation Functions  
- Building models using TensorFlow / PyTorch  
- CNN basics  
- Model training & tuning  

---

## 🤖 Week 3: Generative AI (Gen-AI)  
- Introduction to Generative AI  
- Large Language Models (LLMs)  
- Prompt Engineering  
- Building simple Gen-AI applications  
- Working with APIs and tools
